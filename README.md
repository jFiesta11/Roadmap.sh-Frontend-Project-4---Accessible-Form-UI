# Accessible Form UI

### This project showcases the uses of ARIA Attributes and how it improves accessiblity among all users 

## Features:
* Uses pure HTML 5 and CSS
* Implements **Accessible Rich Internet Application (ARIA)** attributes 
* Includes a custom **Favicon**
* ARIA attributes will be benefitial for users with disabilites, and for users who uses screen readers as their technology assistant 
* Inspect this webpage and click on lighthouse to see an expected overall score of 100 for Accessibility 

## How to Run:

This project can run locally within your computer without installation.

* Clone or Download this Repository.
    ```bash 
    git clone https://github.com/jFiesta11/Roadmap.sh-Frontend-Project-4---Accessible-Form-UI.git
    ```

* Double Click the `index.html` file to open in your default browser or right click the `index.html` file and click on 'open' or 'open with' and select the browser to open this file with.
